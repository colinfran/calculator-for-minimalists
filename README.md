A minimalist calculator app built with **React Native** and **Expo**.  

Designed for simplicity and speed, this app is perfect for quick calculations without distractions. A personal project to practice **Node.js**, **React.js**, and **React Native** skills while building a clean, functional interface.

Check it out live:  
[https://colinfran.github.io/calculator-for-minimalists/](https://colinfran.github.io/calculator-for-minimalists/)

![calculator gif](/assets/calculator.gif?raw=true)

## Features
- Minimalist design
- iOS & Android support
- Smooth, responsive interface
- Built with Expo for easy deployment
